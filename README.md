# Susan Acharya Portfolio

Personal portfolio website for Susan Acharya, Junior Django & Python Developer. Built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

Live site: https://www.susanacharya1.com.np

## Overview

This portfolio showcases:

- Professional summary and introduction
- Technical skills and technologies (Python, Django, React, REST APIs)
- Featured projects section
- Education background and certifications
- Achievements and languages
- Contact section and social links

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui + Radix UI
- React Router

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Run Locally

```bash
git clone https://github.com/susancodex/Susan-Acharya-Portfolio.git
cd Susan-Acharya-Portfolio
npm install
npm run dev
```

Open the local URL shown in the terminal (usually http://localhost:5173).

## Available Scripts

- `npm start`: starts the Vite development server
- `npm run dev`: starts the Vite development server
- `npm run build`: creates a production build in `dist/`
- `npm run preview`: previews the production build locally
- `npm run lint`: runs ESLint

## Project Structure

```text
src/
	components/    Reusable UI and page sections
	contexts/      React context providers
	hooks/         Custom hooks
	locales/       Localization files
	pages/         Route-level pages
	lib/           Shared utilities
```

## Deployment

Build the project:

```bash
npm run build
```

Deploy the generated `dist/` folder to your hosting provider.
